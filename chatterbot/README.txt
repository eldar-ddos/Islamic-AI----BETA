# IslamicBot ChatterBot

IslamicBot is a smart chatbot designed to engage users in meaningful conversations based on Islamic knowledge. 
It uses ChatterBot for core chatbot functionality, Hugging Face Transformers for advanced response generation, sentiment analysis, and web scraping for real-time data.

## Features
- **Smart Responses**: Utilizes DialoGPT-small for generating intelligent and context-aware responses.
- **Offensive Language Filtering**: Includes a built-in filter to ensure appropriate language is used.
- **Sentiment Analysis**: Uses TextBlob to understand user sentiment and tailor responses accordingly.
- **Web Scraping**: Capable of gathering links from various platforms to provide up-to-date information.
